McStay Family Quiz: Documentation

Adding Hero Questions

To add questions for a player, save your JSON content into the specific file for that hero:

  • Patrick: patrick_questions.json
  • Hannah: hannah_questions.json
  • Jillian: jillian_questions.json

Asset Guidelines

  • Images: Max width 800px. Keep under 500KB.
  • Audio: Keep clips under 10 seconds. Use .mp3 format.
  • Naming: Filenames must match JSON entries exactly.

Question Structures

Note: Each question block must be separated by a comma. Do not put a comma after the very last question block in the list.

1. Text Question

{
  "question": "What is the capital of France?",
  "difficulty": "Easy",
  "answers": [
    { "text": "London", "correct": false },
    { "text": "Paris", "correct": true }
  ]
},

2. Visual Question

{
  "question": "Identify this item:",
  "imageSrc": "images/filename.png",
  "answers": [
    { "text": "Correct Answer", "correct": true },
    { "text": "Wrong Answer", "correct": false }
  ]
},

3. Audio Question

{
  "question": "Name this sound effect:",
  "audioSrc": "sounds/filename.mp3",
  "answers": [
    { "text": "Wrong Answer", "correct": false },
    { "text": "Correct Answer", "correct": true }
  ]
}
